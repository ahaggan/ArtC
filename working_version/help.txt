run
used to indicate the start of the program you want to run

{ }
enclose the program instructions

fractal
you can make different types of fractals: tree, sierpinski etc.

iterations
you can choose how many times your fractal gets sub-divided

shape
you can make your fractal out of: lines, circles, squares etc.


colour
you can change the colour of your fractal to: red, blue, green etc.


size
you can change how big your fractal is on the canvas

if and then statements
gives you the option to change the fractal while its generating
